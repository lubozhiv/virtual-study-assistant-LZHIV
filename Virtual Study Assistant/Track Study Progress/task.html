Theory

You’re making great progress! In this stage, you’ll focus on tracking your actual study time and measuring how closely you’ve followed your plan. Tracking progress is essential for staying on target and making adjustments as needed. It also gives you a sense of accomplishment as you complete each session.

By calculating your progress percentage, you’ll learn how to handle real-world scenarios where plans and execution don’t always align perfectly.

Description

Allow users to input the time they have actually studied for each subject. Use this data to calculate their overall progress as a percentage of the planned study time. Provide feedback on how well they’ve done so far.

Objectives

  • Prompt the user to input the actual time studied for each subject

  • Calculate the total time studied

  • Determine the progress percentage by comparing the actual time studied to the planned study time

  • Format percentage value using exactly 2 digits after dot

  • Make sure to clip the percentage value to 100% if the received study time is greater time that expected

Examples

Example 1

Subjects (in memory from previous stages):

subjects = {"Math": 90, "Science": 120}

Input:

Enter time spent studying: 150

Output:

You have completed 71.43% of your planned study time.

Example 2

Subjects (in memory from previous stages):

subjects = {"Math": 90, "Science": 120}

Input:

Enter time spent studying: 210

Output:

You have completed 100.00% of your planned study time.
