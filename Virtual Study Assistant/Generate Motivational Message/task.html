Theory

Now that you’ve tracked your progress, it’s time to make your first call to a real LLM and get some encouragement! Motivational messages can help you stay focused and energized, especially when tackling a challenging study plan. In this stage, you’ll interact with an AI to generate personalized message based on your progress.

To achieve this, you’ll need to interact with the Hugging Face (HF) API. Hugging Face provides a variety of AI-powered tools and models, and accessing them requires creating an API key.

Follow these steps to get started:

  1. Visit Hugging Face and create an account if you don’t already have one.

  2. After logging in, go to your profile (in the top-right corner) and navigate to "Access Tokens".

  3. Click on "+ Generate New Token" button, give it a name, and copy the generated API key.

  4. During key creation, choose "Make calls to the serverless Inference API" option from "Inference" group

  5. Create a .env file in your project directory in PyCharm IDE and paste this token

  6. To read token from secured file, you can use the following code snipped:

    with open('.env', 'r') as fp:
    HF_API_KEY = fp.read().strip()

This API key will allow your code to securely interact with the Hugging Face API without hard-coding sensitive information into your script

Description

Important Note: ⚠️ This project involves the use of private keys and .env files. Please ensure you DO NOT expose your keys by hardcoding them directly into the solution. Instead, always reference them securely from your .env file. Additionally, do not include your .env file in your project submission. Double-check your code for any accidental key exposure before submission. If sharing code snippets, mask sensitive values.

To get your first response from LLM, you’ll use the InferenceClient from the huggingface_hub Python library. For now, we encourage you to explore how this model works and experiment with its parameters. Use the provided code snippet to make your first API call and get hands-on experience with the process.

from huggingface_hub import InferenceClient

client = InferenceClient(token=HF_API_KEY)
prompt = """
I have to prepare for my {subjects} exams. I've completed {completeness:.2f}% of my curriculum. My motivation should be:
""".format(
subjects=','.join(subjects.keys()), 
completeness=completeness
)

response = client.text_generation(
    prompt=prompt,
    model="TinyLlama/TinyLlama-1.1B-Chat-v1.0",
    temperature=0.01,
    max_new_tokens=50,
    seed=42,
    return_full_text=True,
) 
print(response)

Here’s a brief explanation of key parameters you can explore:

  • Temperature: Controls the randomness of the output. Lower values make the output more deterministic, while higher values allow for more "creativity"

  • Prompt: The input text or query sent to the model, guiding the AI on what answer you want to produce

  • Model: The specific pre-trained AI model being used. In this example, we're using one of the smallest LLMs that could produce an interesting result in the shortes time

Please, take into account that it's not comparable to the larges model like OpenAI's GPT-4 but just needed to explore capabilities of open-source models.
