Theory


Congratulations on reaching the final stage! Clean and well-structured code is as important as functionality. Following coding standards like PEP-8 ensures that your code is readable, maintainable, and professional. It’s a valuable habit that will serve you well in your programming journey.

In this stage, you’ll learn to review and refine your code to adhere to PEP-8 guidelines. You’ll also discover tools that can automate this process, saving time and ensuring accuracy.

Description

Review your code for all stages and refactor it to comply with PEP-8 style guidelines. You can simply push the "Check" button to automatically validate your code. This will provide you with suggestions for improvement and evaluate your work on a 10-point scale. Use these insights to refine your code to meet professional standards.

Objectives

  • Analyze your code for any deviations from PEP-8 standards.

  • Ensure the final version of your project is clean and professional.

  • Hint: You can use automated tools (black, pylint) to format and validate your code.
